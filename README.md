# Spotify Web Player Clone

This is a simple frontend clone of the Spotify Web Player. I made this project using HTML and CSS to practice layout design, styling, flexbox, and building a real-world looking user interface.

The project is not fully functional like the original Spotify app, but it focuses on recreating the visual design and basic structure of the Spotify web player.

## Features

- Spotify-style sidebar navigation
- Home and Search section
- Library section with playlist and podcast cards
- Sticky top navigation bar
- Music cards for recently played, trending, and featured charts
- Fixed bottom music player
- Responsive behavior for smaller screen sizes
- Custom styling using CSS
- Icons using Font Awesome
- Google Font integration

## Tech Stack

- HTML5
- CSS3
- Font Awesome
- Google Fonts

## How to Run This Project

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No extra installation is required because this is a static frontend project.
